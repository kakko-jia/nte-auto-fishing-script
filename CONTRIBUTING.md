# Contributing

Thanks for improving this project.

## Development Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Before Opening a Pull Request

- Keep changes focused and easy to review.
- Test the script on Windows.
- Mention your screen resolution and game window mode if the change affects detection.
- Avoid committing local logs, screenshots, virtual environments, or personal configuration.
